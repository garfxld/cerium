# Cerium

Yet another high-performance Minecraft server library written in Rust.

> [!CAUTION]
> This repository is still in a very, very, very early stage of development.


## Goals

- High-performance
- Lightweight
- Easy to use


## Roadmap

- Protocol
    - [x] Server list ping
    - [x] Encryption
    - [x] Compression
    - [ ] Joining a world (WIP)
    - [x] Registries
- World
    - [ ] Entities
    - [ ] Block Interactions
- [ ] Items
- [ ] Inventories
- [ ] Event system (WIP)

Of course, more features are planned for the future.


## Running

```sh
cargo r --example flat_world
```
